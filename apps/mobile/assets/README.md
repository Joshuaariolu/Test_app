# Assets Directory

This directory contains the app assets:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## Placeholder Assets

The current assets are placeholders. Replace them with actual app branding assets when available.

## Asset Requirements

- **Icon**: 1024x1024 PNG with transparent background
- **Splash**: 1284x2778 PNG for iPhone 12 Pro Max
- **Adaptive Icon**: 1024x1024 PNG with 192px safe area
- **Favicon**: 32x32 PNG for web
